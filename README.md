基于成渝地区租房信息的数据分析
====
  **主要内容：抓取链家上成渝地区核心区域租房信息，分析两地核心区房价排名、价格区间分布和户型和面积分布情况**
  <br/>**主要实现包**：requests、pymongo、pyquery、pandas、pyecharts<br/>
  <br/>**其中**lianjia.py为成渝租房信息的爬虫代码，rent1.csv为抓取结果，rent_cy为数据分析过程（数据可视化采用包为pyecharts，在github上不能直接显示，通过jupyter notebook可打开）

### 2.明确问题与数据分析
#### 2.1  成渝核心区房价排名情况
* 重庆租金现状分析
  * 按单位面积月租金统计，重庆地区平均租金在25.21元/(㎡/月），其中渝中区租金最高，江渝北租金接近，位列第二第三。原因在于：从总体上看，这三个区域对应的GDP是重庆市最高的，经济创造价值高的区域对应的人口活跃度高，对应的房价也高。从各区域定位看，渝中地区是重庆市的历史人文母城、政治中心，也是地理几何中心；江渝北地区是重庆就业中心，也是两江新区的核心区域，两江新区是国家级新区，重庆最重要的经济增长极，其中江北地区是重庆金融行业的聚集地，对应的劳动人口需求量大、薪资高，支撑起了该区域的租金；而渝北区是重庆企业数量最多的聚居地，也是就业人口最多的地区，不过该地区的面积较为广阔，大部分地区还发展未成熟，因此平均房价仅位于第三，相信随着时间的推移，城区发展成熟，未来的租金水平将会上升。

* 成都租金现状分析
  + 按单位面积月租金统计，成都地区平均租金在30.20元/(㎡/月），其中高新区租金最高，武侯、锦江区租金相同，为第二第三。原因在于：高新区是成都的就业中心，区域交通成熟，包含天府软件园在内的众多提供就业机会的聚集地，是成都新增人口首选区域，就业人员平均薪资水平高，这些因素促使该区域租赁房价处于首位。而武侯、锦江区是成都的传统核心区，处于成都中心城区一圈，设施配套成熟、交通便利，又临近高新区，因此租金水平也较高。

* 对比分析
  + 成都地区租金高于重庆，包含成都薪资水平高于重庆，且成都地区净流入人口数量高，对租房的需求量更大等因素

#### 2.2  结合价格区间分布情况分析
* 将成渝地区房屋租金价格按照[0,500,1000,1500,2000,3000,5000,10000,100000]划分，统计各区间房屋数量作折线图，发现成渝两地各区间房屋数量趋势接近，租金处于(500，1000]房屋数量最多，其次在于(2000,3000],高于3000的租房数量较少且呈递减趋势，表明了成渝地区租金水平较低的现状。原因在于，1、成渝地区整体薪资价格水平较低。2、租金处于(500，1000]租赁房屋类型多为单间(2000,3000]多为整租。

#### 2.3  户型和面积分布情况
* 户型分布情况
  + 考虑到不同租赁人对房屋户型的需求不同，本人通过对成渝地区核心区租房数据的户型按照**整租和合租**两个租赁类型进行分组统计，展示户型数量总体中占比较高的类型，用Pyecharts饼图展示得到户型分布图。 统计发现，整租类型的租赁人，其中受欢迎的租房类型为3室2厅、2室1厅、1室1厅。其中选择3室2厅和2室1厅的人员理应为家庭，占比43.5%。选择1室1厅多为未婚，占比13.15%，两者占比超50%。合租类型的租赁人，其中受欢迎的租房类型为4室1厅、5室1厅和3室1厅，和占比超过80%，究其原因离不开房价更低。

* 面积分布情况
  + 通过划分区间[0,30,60,90,120,150,200,300,1000]对各种户型的面积进行分组，统计各区间房间面积数量，通过玫瑰图展示各占比如图所示。 统计发现，面积越大，出租房屋数量越少。其中选择面积为(0,30]的房屋最多，占比32.85%，多为未婚人士的选择，(30,60],(60,90]在2、3位，和占比超40%，多为已婚人士的选择。结合两类人员的租赁选择事实，表明了租赁者倾向于选择小户型。

### 3.结论
  抓取的成渝地区租房信息，通过分析两地核心区房价排名、价格区间分布和户型和面积分布情况，得出以下结论：<br/>  
* 成都地区高新区、武侯区和锦江区租价最高，重庆地区渝中区、江北区和渝北区租价最高，成都地区租金水平整体高于重庆。
* 成渝地区租金水平较低，多处于(500，1000]。
* 对于选择整租的客户而言，小户型更受青睐；对于选择合租的客户而言，多居室户型有更多选择。
